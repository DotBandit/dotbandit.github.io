DotBandit

DotBandit is an Open-source Administration Tool and Utility for Windows. It is able to build executable files by customizing a few options. Log reports are generated into an HTML file with a generated table containing Active windows opened and Keystrokes typed.


Features

  • Executable Builder
  • Keystrokes recording
  • Active Window grabbing
  • Generated .html report
  • IP grabbing and related information
  • Adding to Startup
  • Auto Screenshots (7 different image qualities)
  • MessageBox customization
  • Add-ins (implementing own code)
  • Password protected application

System Requirements

  • DotBandit runs on the Microsoft .NET Framework 3.5
